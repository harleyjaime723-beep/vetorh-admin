⚙️ Gestão de Acessos
Administradores autorizados a aceder ao painel
Como funciona
Os acessos são controlados directamente no código backend (adminDashboard.ts). Para adicionar ou remover um administrador, edita a lista ADMIN_EMAILS e reimplementa a função.

A senha é definida pela variável de ambiente ADMIN_PASSWORD no Base44. Se não estiver definida, usa a senha padrão temporária.
Admins actuais
🔐 Alterar Senha
Para alterar a senha, define a variável ADMIN_PASSWORD nas definições do Base44 → Backend Functions → Environment Variables.
➕ Adicionar Admin
Edita o ficheiro functions/adminDashboard.ts no Base44 App Builder e adiciona o email à lista ADMIN_EMAILS. Depois reimplementa.
const ADMIN_EMAILS: string[] = [
  "admin@vetorh.ao",
  "harley@vetorh.ao",
  // adiciona aqui os novos admins
];
